# Snake Game

Snake Game is a JavaScript version of the arcade game ***Blockade*** released in **1976** by [Gremlin].
It uses MooTools JavaScript library.

* ## Elements
    * ### Canvas
        * **Properties**
            * Size
            * Background Color
    * ### Snake
        * **Properties**
            * Size
            * Length
            * Color
    * ### SnakeFood
        * **Properties**
            * Size
            * Color
